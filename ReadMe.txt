1. Open Blind_Eye Template.xlsm File
2. By default, you will be having Maker Access
3. Press Alt + F8
4. Select devUnlock and Click on Run
5. Enter "dev" without quotes as password
6. All the Sheets will be unlocked
7. Go to Main Sheet, You need to enter NT ID in Column A, Name in Column B, and Role in Column C
8. Your NT ID will be visible in Cell E1 of Main Sheet
9. You can copy value from Cell E1 and paste in Column A to define your role

Once you define role, you do not need to activate Dev mode, simply save and close the file and re-open the file, macro will ask for password to access Main Sheet
